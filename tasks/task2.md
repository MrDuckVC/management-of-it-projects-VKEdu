# Практическое задание №2

* Определите роли и напишите User Stories: Выделите целевые аудитории продукта из первого задания. Сформулируйте 10–20 историй для основного пути пользователя.
* Приоритезируйте функционал: Оцените получившийся список историй любым фреймворком на выбор: MoSCoW, RICE или Kano.
* Сформируйте MVP и MLP: Разделите истории. Отделите критически важное для работы («скелет» MVP) от фич, дающих вау-эффект и удобство (MLP).
* Детализируйте требования: Выберите 2 самые главные истории из MVP. Пропишите для них строгие Функциональные (ФТ) и Нефункциональные (НФТ) требования.
